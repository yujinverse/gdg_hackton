## 📌 Problem

- 무엇을 준비해야 할지 모름
- 계획은 있지만 실행이 어려움
- 이력서 수준을 객관적으로 알기 어려움
- 나에게 맞는 기업 찾기 어려움

---

## 💡 Solution

알짜배기는 이력서를 기반으로

1. AI 역량 분석
2. 부족한 부분 태스크 생성
3. 일정 캘린더 로드맵 제공
4. 맞춤 기업 추천

👉 **취업 준비를 ‘행동 계획’으로 변환**

---

## ✨ Features

### 📝 Resume
- 이력서 업로드
- AI 요약/피드백
- 상세 분석 리포트
- 추천 면접 질문

### 📅 Roadmap
- AI Task 자동 생성
- 일정 캘린더 설계
- 단계별 학습/준비 가이드

### 🏢 Company
- 맞춤 기업 추천
- 관심 기업 관리
- 채용공고 확인

### 🔐 Auth
- 로그인 / 회원가입
- 사용자 프로필 관리
